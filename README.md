# touch-for-windows
A Unix tool rewritten in C++ for the Windows operating system.

![](https://github.com/0xmessier/touch-for-windows/blob/main/img/Screenshot_1.png?raw=true)

# Build & Install
```
> build.bat
> install.bat (with highest privilege)
```

# Run
```
> touch
Usage:
  touch [filename]
```
